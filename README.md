# Secure Network Services & VPN Configuration Lab

## Overview
This project documents a practical network security lab using Kali Linux and Ubuntu virtual machines. The lab covered host connectivity testing, packet capture, service enumeration, FTP/SFTP traffic comparison, firewall testing, OpenVPN setup, certificate generation, VPN tunnel verification, and SSH key-based authentication.

## Objectives
- Verify connectivity between Kali and Ubuntu machines.
- Capture and analyze ICMP, FTP, SFTP, and VPN traffic using Wireshark.
- Perform host discovery and service enumeration using Nmap.
- Compare plaintext FTP traffic with encrypted SFTP traffic.
- Configure and test UFW firewall rules.
- Install and configure OpenVPN with Easy-RSA.
- Generate CA, server, and client certificates.
- Establish an encrypted VPN tunnel.
- Configure SSH key-based login and disable password authentication.

## Tools Used

## Key Learning Outcomes
- Network traffic analysis
- Service enumeration
- Secure remote access
- VPN tunnel configuration
- Firewall rule validation
- SSH hardening
- Understanding the difference between plaintext and encrypted protocols

## Disclaimer
This project was completed in a controlled virtual lab environment for educational purposes only. All sensitive keys and configuration secrets have been removed.
